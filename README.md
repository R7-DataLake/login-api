# Login API

Environments: 

```
R7_LOGIN_DB_HOST=localhost
R7_LOGIN_DB_PORT=5433
R7_LOGIN_DB_NAME=xxxxx
R7_LOGIN_DB_SCHEMA=xxxx
R7_LOGIN_DB_USER=xxxx
R7_LOGIN_DB_PASSWORD=xxxxxx

R7_LOGIN_PORT=3021
R7_LOGIN_SECRET_KEY=xxxxxxxxx

NODE_ENV=development
```

# Run

```
NODE_ENV=development \
R7_LOGIN_DB_HOST=localhost \
R7_LOGIN_DB_NAME=xxxxx \
R7_LOGIN_DB_SCHEMA=xxxx \
R7_LOGIN_DB_USER=xxxx \
R7_LOGIN_DB_PASSWORD=xxxxxx \
R7_LOGIN_SECRET_KEY=xxxxxxxxx \
R7_LOGIN_PORT=3021 \
npm start
```

# Build

```
npm run build
```
