# Login API

Environments: 

```
R7PLATFORM_LOGIN_DB_HOST=localhost
R7PLATFORM_LOGIN_DB_PORT=5433
R7PLATFORM_LOGIN_DB_NAME=xxxxx
R7PLATFORM_LOGIN_DB_SCHEMA=xxxx
R7PLATFORM_LOGIN_DB_USER=xxxx
R7PLATFORM_LOGIN_DB_PASSWORD=xxxxxx
R7PLATFORM_LOGIN_DB_POOL_MIN=0
R7PLATFORM_LOGIN_DB_POOL_MAX=500
R7PLATFORM_LOGIN_PORT=3000
R7PLATFORM_LOGIN_SECRET_KEY=xxxxxxxxx
R7PLATFORM_LOGIN_SEND_SECRET_KEY=yyyyyyyyyyyy

NODE_ENV=development
```

# Run

```
NODE_ENV=development \
R7PLATFORM_LOGIN_DB_HOST=localhost \
R7PLATFORM_LOGIN_DB_NAME=xxxxx \
R7PLATFORM_LOGIN_DB_SCHEMA=xxxx \
R7PLATFORM_LOGIN_DB_USER=xxxx \
R7PLATFORM_LOGIN_DB_PASSWORD=xxxxxx \
R7PLATFORM_LOGIN_SECRET_KEY=xxxxxxxxx \
R7PLATFORM_LOGIN_SEND_SECRET_KEY=xxxxxxxxx \
R7PLATFORM_LOGIN_PASSWORD_SALT=xxxxxxx \
npm start
```

# Build

```
npm run build
```
